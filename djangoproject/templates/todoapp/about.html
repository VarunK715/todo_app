Usage Instructions

Adding Tasks

  1. Log in to the application.
  2. Use the form to add new tasks, specifying the task name, priority, and due date.
  3. Click the "Add Task" button to save the task.

Managing Tasks

  • Update: Click the "Update" button next to a task to modify its details. The update button is disabled for completed tasks.
  • Delete: Click the "Delete" button next to a task to remove it. Completed tasks are automatically deleted.
  • Mark as Completed: Use the checkbox to mark a task as completed. Once marked, the task will be automatically deleted.

Sorting and Searching

  • Sort by Priority: Use the dropdown menu to sort tasks by priority level.
  • Search Tasks: Use the search bar to find tasks by keyword.

Due Date Warnings

  • If a task's due date is today or tomorrow, a warning message will be displayed.
  • If the task is not completed by due date, it will be deleted automatically. You have a option to update the due date by clicking on update button.

Authentication

  • Registration: Create a new account using the registration page.
  • Login: Log in to access your tasks and profile.
  • Logout: Log out to end your session.
  • Profile: View your profile details.
